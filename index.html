Personal Website

Who was it for?

This website was created for myself and for anyone willing to follow along.

What was it?

This project is my own personal website that I have been slowly working on. It was first created February 2015 and I have been slowly modifying the layout, fixing mobile issues, adding new pages, and trying to create a real theme for myself. It was a project done in C++ to implement our own Integer class.

When was it?

First created in February 2015 and has had a continuous string of change every week since.

Where was this?

This was done from home during my leisure.

Why was it done?

I have always wanted my own personal website where I could project my thoughts, my style (music and ideas), for anyone who was interestd and show my love for photography.

How was it done?

By using Wix's HTML editor, I slowly created my own website. I am still learning about adding my own HTML code and changes in CSS style files as I go.
